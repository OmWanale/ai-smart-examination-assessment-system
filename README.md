# AI Smart Examination & Assessment System

An AI-powered smart examination platform supporting:

- Teacher & Student authentication
- Class creation & joining
- Manual quiz creation
- AI quiz generation (Gemini API)
- Quiz attempts with timer
- Results & analytics
- Explanation review system
- Desktop app via Electron
- Modern UI with dark mode

## Tech Stack
- React
- Node.js
- Express
- MongoDB Atlas
- Gemini AI API
- Electron

## Features
- Role-based dashboards
- AI quiz auto-generation
- Result permission control
- Class code joining
- Teacher analytics
